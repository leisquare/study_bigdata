
[02] 변수

	1. 변수
		프로그램 작업 처리를 위한 하나의 값을 저장할 수 있는 메모리 공간. 혹은 그 공간에 이름을 붙인 것.
		자바에서는 한 가지 타입을 한 가지 변수에 저장 가능.
		식별자(identifier)라고도 부른다.

	2. 기본 데이터 타입
		1) 논리형- boolean 1byte 표현 범위는 true, false
		2) 문자형- char 2byte 
		3) 정수형-byte 1byte
			-short 2byte
			-int 4byte
			-lont 8byte
		4) 실수형-float 4byte
			-double 8byte
		
	3. 기본 데이터 타입과 참조 데이터 타입의 이해
		1) 기본 데이터 타입: 메모리에 있는 실제 값=변수 데이터 값. 주로 간단한 데이터들
		2) 참조 데이터 타입: 메모리에 있는 실제 값=변수가 저장된 주소. 실제 변수 데이터는 heap 영역
				기본 자료형에 비해서 크기가 크다.

	4. 형변환
		1) 묵시적 형변환: 작은 주머니의 데이터를 큰 주머니로. 
		2) 명시적 형변환: 큰 주머니의 데이터를 작은 주머니로. 데이터 손실이 있을 수 있음. 코드에 명시한다.

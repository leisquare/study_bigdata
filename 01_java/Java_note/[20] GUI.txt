[18] 자바 GUI(Graphic User Interface)

	1. Graphic User Interface란?
		사용자가 프로그램을 사용하기 편리하도록 그래픽 요소를 가미해 만든 인터페이스. 

	2. AWT 컴포넌트
		AWT: JAVA에서 그래픽 요소를 만들기 위한 컴포넌트. 예컨대 버튼, 체크박스 등이 있다.
		1) 컴포넌트 배치와 관련: BorderLayout, GridLayout, FlowLayout
		2) GUI 구성과 관련 : Button, Label, Canvas, Checkbox, Choice, Scrollbar, List, Menu, TextComponent, TextArea, TextField, CheckboxGroup	
		3) 컨테이너 클래스 등등 : Frame, Panel, Window, Container, FileDialog, Demension, Event

		*AWT의 사용법
		컴포넌트 생성(객체 선언, 컴포넌트 부착)-객체 생성-이벤트 처리
	
	2. Swing 컴포넌트
		AWT보다 진보된 기능을 제공. 
		
		*Swing의 사용법
		1) 패널 또는 컨테이너 만들어 레이아웃 세팅
		2) 컴포넌트 추가 및 크기 확인
		3) 패널을 프레임에 추가
		4) 패널의 크기 확인
		5) 크기 수락 여부 결정

[05] Array 

	1. 배열이란?
		동일 자료형의 집합. 하나의 이름으로 여러 데이터 사용 가능.	
		여러 변수(주머니)를 모은 또 하나의 큰 주머니.

	2. 배열 문법
		변수 선언과 비슷하게 선언하되, 여러 데이터가 모여있어 {}을 이용한다.
		단, 배열의 크기는 최초의 설정에서 변경할 수 없다.
		배열은 객체로 취급된다.
		
	3. 배열 메모리 구조의 이해
		1) int i=10;  메모리에 i 주머니가 있고, 주머니 안에 10이 있다.
		2) int[] iArr = {10,20,30}; 메모리에 iArr 주머니가 만들어지고, iArr 주머니 안에는 배열을 구성하고 있는 데이터의 주소값이 들어있다.
	
	4. 레퍼런스의 이해
		1) 배열을 구성하는 데이터들의 주소값을 '레퍼런스'라고 함
		2) 레퍼런스가 동일하다면 같은 데이터를 가리키는 것.

	5. 다차원배열
		배열 안에 또 다른 배열이 들어있는 구조.

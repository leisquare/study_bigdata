
[02] HTML-1

	1. 태그란?
		1) HTML 문서를 구성하는 요소. 여는 태그, 닫는 태그로 나뉜다.
		태그에는 이름과 속성, 속성값이 있다.
		ex) <a href="http://www.daum.net"> 태그 이름 : A, 태그속성 :href, 태그 값 순

	2. html 구조
		1) <!DOCTYPE html>현재 웹페이지가 HTML5 문서임을 인식하게 만들어줌.
		2) <html  lang=”ko”> 태그의 시작이자 lang 속성을 입력 가능. 
		3) <head> - </head> 타이틀이나, 설정값인 CSS, javaScript 등의 정보를 넣는다.
		들어갈 수 있는 태그는 meta, title, script, link, style
		4) <body> - </body> 사용자에게 보이는 웹문서 실제 정보.

	3. 글자 관련 태그
		1) h1~h6 : 헤더를 의미하는 태그. 개행을 포함함.
		2) p : <p>주로 본문에 사용되는 태그.
		3) /br : 개행
		4) a : 앵커. 어디로 이동할지는 href 속성값으로 지정한다. 경로를 프로토콜까지 입력할 것. 
		5) i : 이탤릭
		6) sup, sub : 윗첨자 아랫첨자
		7) ins : 언더라인
		8) del : 취소선

	4. 목록 태그
		1) ul: 순서가 없는 목록 태그
		2) ol: 순서가 있는 목록 태그
		3) li : 목록 요소
		4) dl : 정의 목록 태그
		5) dt : 정의 용어 태그
		6) dd : 정의 설명 태그

	5. 표(table) 태그
		1) tr : 표 내부 행 태그
		2) th : 행 내부의 열. 제목 셀 태그. 자동적으로 가운데 정렬이 됨.
		3) td : 행 내부의 열. 일반 셀 태그
		* 행, 열을 병합하고싶을때는 colspan, rowspan을 사용한다. (ex <td rowspan="3">)

	6. 이미지 태그
		1) src : 이미지 경로 지정
		2) alt : 이미지없을때 나올 글자 지정
		3) width : 너비 지정
		4) height : 높이 지정
		* 너비, 높이는 CSS에 지정하는 경우가 많다.

	7. 오디오 태그
	8. 비디오 태그
	9. detail, summary
		1) detail : 세부정보나 추가 정보를 숨겨놓았다 펼치는 기능
		2) summary : 세부사항의 제목 또는 요약 내용

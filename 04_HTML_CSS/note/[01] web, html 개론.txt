
[01] web, html 개론

	1. web 이란?
		-1개 이상의 사이트가 연결되어 있는 인터넷 서비스의 한 가지 형태. 
		-프로토콜: 네트워크상에 약속된 퉁신규약.
		-IP: 네트워크상에서 컴퓨터를 식별할 수 있는 주소
		-DNS: IP 주소를 인간이 쉽게 외우도록 맵핑한 문자열
		-Port: 해당 컴퓨터의 구동되고 있는 프로그램을 구분할 수 있는 번호.

		*웹 서버- 웹브라우저의 요청을 받아 결과를 웹 브라우저에 전송하는 역할. 
		프로그램 처리가 필요하면 애플리케이션 서버를 사용하거나 프로그램을 호출해서 처리. 

		* 플러그인: 웹브라우저 전쟁 때 발생한 기술을 제대로 표준화하지 못해 불만을 느낀 기업들이 개발한 기술. 
		액티브 엑스, 플래시 등이 있음. 웹 브라우저와 연동되는 특정 프로그램을 
		사용자 PC에 추가로 설치, 웹브라우저 기능을 확장하는 방법. 

	2. html이란? 
		Hyper Text Markup Language. 웹문서를 기술하는 언어.
		컴퓨터에서 사용하는 모든 파일에는 각각 고유의 형식이 있다. 
		그 중 웹에서 텍스트, 이미지, 링크 등 여러 요소를 다루고 표시할 수 있는 문서를 작성하는 언어가 HTML

	3. CSS란?
		Cascading Style Sheets, HTML문서의 디자인 파트로, 정보 전달을 좀더 효율적으로 하기 위해 만들어진 문서.
		HTML이 정보 부분, CSS가 화면상 레이아웃 컨트롤.

	4. JAVAScript-jQuery
		동적인 부분을 작성

	5. 기본적 웹 표준 문서의 상호관계.
		-웹표준은 웹문서의 구조와 표현, 동작을 각각 구분한다.
		1) HTML: 문서의 구조, 정보
		2) CSS: 표현(디자인)
		3) Java Script: 동작
